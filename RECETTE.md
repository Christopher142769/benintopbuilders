Utiliser les comptes du `npm run seed` (mot de passe membres : `Password123!`).

Landing → S'inscrire → profil → infos + RCCM PDF → OTP (logs API) → charte → palier Découverte → dashboard.

## 2. Visibilité + label
Ma fiche → publier → Demander labellisation → pièces + paiement → admin examine → badge.

## 3. Appels d'offres
Publier AO → Premium voit tout de suite / Standard après 24h → réponse → Étudier/Retenir → messagerie.

## 4. Marketplace
Catalogue → panier → adresse → FSPay sandbox → fournisseur : en préparation → livrée → avis.

## 5. Formations
Catalogue → inscription multi-participants → paiement → admin émargement → attestation PDF.

## 6. Messagerie
Contacter depuis fiche → chat temps réel → masquage contacts si un participant non actif.

## 7. Cycle adhésion
Dashboard adhésion → renouvellement auto → historique paiements → (crons J-15 / J-0 / J+7).

## 8. Admin
`/admin` → dossiers, membres, moderation (avis / prix / doublons AO), stats, audit.
